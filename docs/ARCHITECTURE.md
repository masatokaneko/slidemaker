# アーキテクチャドキュメント

## システム概要

SlideMakerは、自然言語からPowerPointプレゼンテーションを生成するAIツールです。システムは以下の主要コンポーネントで構成されています：

1. フロントエンド（Next.js）
2. バックエンドAPI（Next.js API Routes）
3. データベース（PostgreSQL）
4. キャッシュ（Redis）
5. ファイルストレージ
6. モニタリングシステム
